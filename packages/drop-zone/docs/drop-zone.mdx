# Dropzone

<Metadata pkg={pkg}/>

Drag and drop of file uploading

## Dropzone with `innerZone`

Modify the content of dropzone with `innerZone` props.

<Playground>
    <div style={{width: 200, margin: 'auto'}}>
        <DropZone 
            onResult={(url) => console.log(url)}
            innerZone={
                <div style={{width: '100%', height: 150, display: 'flex', justifyContent: 'center', alignItems:'center'}}>
                    <TextDelta color={active} style={{textAlign: 'center'}}>Custom dropzone</TextDelta>
                </div>     
            } 
        />
    </div>
</Playground>

## `file` props

Pass a file to the DropZone through `file` props to tell DropZone to handle the file. If `onlyImage` is `true`, a preview will be shown.
