#### Props
##### `loadOptions` - async function that returns an array of hashtags in format `[{hashtag: 'test', count: 33}]`
##### `callback` - a function called when new hashtag is added/created. Returns an array of currently selected hashtags: `[{hashtag: 'test'}]`
##### `defaultValue` - pre-selected values for the input in format `[{hashtag: 'test'}]`
