## Borders

Function `border(type: String)` receives the name of the border and returns the corresponding color.
```

const Box = styled.div`
  width: 100%;
  height: 40px;
  border-radius: ${radius400};
  border: ${border("gray")};
`;
```

## Animations

Animations have five basic animations `dropFade`, `fadeInOut`, `fadeIn`, `scaleAndFadeIn`, `scaleAndFadeOut`.
There is one important constants `animationCurve`, which is our basic animation curve (duh). You can also use "linear" or othes that work for the occasion.
The 2 main speeds for animations are `animationFast`, which is 260ms and `animationSlow` which is 600ms.

```
export const ElementDropFade = styled.div`
  width: 40px;
  height: 40px;
  border-radius: ${radius400};
  background-color: green;
  animation: ${dropFade} ${animationFast} ${animationCurve} infinite;
`;
```
