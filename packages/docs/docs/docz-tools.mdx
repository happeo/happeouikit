# Docz Tools

Some handy tools for documentation to represent.

There is also `Playground` and `Props` available by docz internally, see their documentation here: [Docz Component Api](https://www.docz.site/documentation/components-api).

<Metadata pkg={pkg} />

## Metadata

<Props of={Metadata} />

This needs to be included into all documentation files of packages. See the example at the top of this document.

```js
import pkg from "../package.json";
<Metadata pkg={pkg} />
```

## MarginExample

<Props of={MarginExample} />

Use "Shorthand property names" to get the variable name as padding name.

```js
const hugeMargin = "40px";
<MarginExample margin={{hugeMargin}} />
```

> Note: Names and values below are not from our guidelines.

<Playground>
  <MarginExample margin={{tinyMargin: "4px"}} />
  <MarginExample margin={{hugeMargin: "40px"}} />
</Playground>
